# Project 13 – Evading IDS, Firewalls and Honeypots

## Overview

This project is about learning how to protect computer networks from attacks by using Firewalls, Intrusion Detection Systems and Honeypots. We will be using **iptables** on Ubuntu Linux to practice setting up Firewalls. The project will cover what Firewalls, Intrusion Detection Systems and Honeypots do and how they work together to keep networks safe. We will also learn about the role of a Security Operations Center analyst.

The project has a part where we will learn how to set up Linux firewall rules to control network traffic and make the system more secure. At the end of the project we will learn about the ways to protect networks and how to respond to security incidents.

> **Disclaimer:** All the work in this project was done in a controlled laboratory environment for educational purposes only.

---

## Objectives

- Learn what Firewalls, IDS, IPS and Honeypots are used for.

- Set up firewall rules using **iptables** on Ubuntu.

- Control network traffic based on ICMP, ports and IP addresses.

- Learn about Defense-in-Depth strategies.

- Learn practices for Firewalls and IDS.

- Understand what a Security Operations Center analyst does.

- Learn about the incident response process.
